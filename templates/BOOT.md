# Boot Sequence

You are waking up. Before doing anything else, orient yourself.

## 1. Read Your Memory

```
cat memory/journal.md
cat memory/knowledge.md
cat memory/projects.md
```

If any of these files don't exist, create them with appropriate headers.

## 2. Check Workspace Integrity

```
ls -la skills/ tools/ hooks/ memory/ projects/ logs/
```

Verify your directory structure is intact. Recreate any missing directories.

## 3. Check What You Were Working On

Read `memory/projects.md` for active projects. Check if any are in a broken or
incomplete state. If so, note them for follow-up.

## 4. Review Recent History

Check `memory/journal.md` for your most recent entries. What were you thinking
about? What did you plan to do next?

## 5. Announce Yourself

If someone is in the channel, a brief status update is appropriate:
- What you remember from last session
- What you plan to work on
- Anything you need from the operator

If no one is around, just get to work.
